DecodeLabs Project 1 – Advanced EDA & Feature Engineering

Project Overview
This project was completed as part of the DecodeLabs Data Science Internship Program. The objective is to transform a raw dataset into a clean, machine learning-ready dataset using Exploratory Data Analysis (EDA), statistical data cleaning, and feature engineering techniques.

Objectives
- Perform Exploratory Data Analysis (EDA)
- Handle missing values
- Detect and treat outliers
- Create new predictive features
- Apply vectorized computations
- Validate and scale data
- Prepare a clean dataset for machine learning

🛠️Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Pandera
- Google Colab

Files Included
- Project1.ipynb
- Cleaned_Dataset.csv
- Central_Feature_Store.csv

Project Workflow

Phase 1: Advanced EDA & Feature Engineering
- Loaded the dataset
- Performed Exploratory Data Analysis
- Handled missing values
- Detected and removed outliers using IQR
- Created new predictive features

 Phase 2: Vectorized Computation Engine
- Performed vectorized operations using NumPy and Pandas
- Applied Label Encoding and One-Hot Encoding
- Analyzed feature correlations

Phase 3: Structural Contracts and Scaling
- Validated data using Pandera
- Applied StandardScaler
- Created a Central Feature Store
- Ensured data consistency for machine learning

Outcome
Successfully transformed the raw dataset into a clean, validated, and feature-engineered dataset ready for machine learning applications.
